# README

* Ruby version: 2.3.4
* Rails version: 5.1.4

## Gems needed to run: 
  *`google_places`
  
  *`geocoder`
  Add the following to your Gemfile and run 'bundle install'
  `gem 'google_places'
  
   gem 'geocoder' `

* How to run the test suite
  To run tests add these gems:
  * `rspec-rails`
  * `faker`
  * `simplecov`
  * `autotest-rails`
  * `factory_bot_rails`
  Add the following to your Gemfile and run 'bundle install'
   `gem 'rspec-rails'
    gem 'faker'
    gem 'simplecov'
    gem 'autotest-rails'
    gem 'factory_bot_rails'`
